NTPU_MIS
=====================


## key setting
1. coffee script
2. reactjs
3. browserify
4. compass

# Install

```shell
npm install
```

# How to run it?

```shell
gulp server
```
and open browser <http://localhost:3000>

# Test

```shell
gulp test
```

# Build project

```shell
gulp build
```

# Build project as production

```shell
NODE_ENV=production gulp
```
